# JF Design Patterns

常见设计模式有哪些：

- [单例模式 Singleton Pattern](DesignPatterns/singleton.md)
- [备忘录模式 Mememto Pattern](DesignPatterns/mememto.md)
- [工厂模式 Factory Pattern](DesignPatterns/factory.md)
- [策略模式]
- [代理模式]
- [装饰模式]
- [观察者模式]
